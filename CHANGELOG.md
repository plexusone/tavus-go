## [v0.1.0] - 2025-07-06 ([`3733f84`](https://github.com/plexusone/tavus-go/commit/3733f84))

### Highlights

- Initial release of Go SDK for Tavus API with full CVI (Conversational Video Interface) support
- Spec-driven development using OpenAPI 3.0.3 and ogen code generation
- High-level service wrappers for all 10 Tavus resource types
