# Release v0.1.0

**Release Date:** 2025-07-06

Initial release of the Go SDK for the [Tavus API](https://docs.tavus.io/).

## Overview

tavus-go provides a comprehensive Go client for Tavus's Conversational Video Interface (CVI) platform. This SDK enables developers to integrate AI-powered video avatars, real-time conversations, and video generation into Go applications.

## Highlights

- **Full CVI Support** - Complete coverage of Tavus API v2 endpoints for conversational video
- **Spec-Driven Development** - Built from OpenAPI 3.0.3 spec using ogen code generation
- **Type-Safe API** - Strongly-typed request/response handling with proper error types
- **10 Service Wrappers** - High-level interfaces for all Tavus resource types

## Features

### Services

| Service | Description |
|---------|-------------|
| `ConversationsService` | Real-time video session management |
| `PalsService` | Personality AI Layer (PAL) configuration |
| `FacesService` | Visual identity / avatar management |
| `VoicesService` | Stock voice listing |
| `VideosService` | Async video generation |
| `ToolsService` | Function calling tool management |
| `GuardrailsService` | Behavioral boundary configuration |
| `ObjectivesService` | Conversation goal management |
| `DocumentsService` | Knowledge base document management |
| `DeploymentsService` | Distribution channel management |

### Client Options

```go
// Environment variable (default)
client, _ := tavus.NewClient()

// Explicit API key
client, _ := tavus.NewClient(tavus.WithAPIKey("your-key"))

// Custom configuration
client, _ := tavus.NewClient(
    tavus.WithAPIKey("your-key"),
    tavus.WithBaseURL("https://custom.tavusapi.com"),
    tavus.WithTimeout(60 * time.Second),
    tavus.WithHTTPClient(customHTTPClient),
)
```

## Quick Start

```go
package main

import (
    "context"
    "log"

    "github.com/plexusone/tavus-go"
    "github.com/plexusone/tavus-go/internal/api"
)

func main() {
    client, err := tavus.NewClient()
    if err != nil {
        log.Fatal(err)
    }

    ctx := context.Background()

    // Create a conversation
    conv, err := client.Conversations().Create(ctx, &api.CreateConversationRequest{
        PalID:  "pal_abc123",
        FaceID: "face_xyz789",
    })
    if err != nil {
        log.Fatal(err)
    }

    log.Printf("Conversation URL: %s", conv.ConversationURL.Value)
}
```

## Examples

The SDK includes several examples demonstrating common use cases:

- **basic/** - Conversation creation and management
- **list-resources/** - Listing all resource types
- **pal-management/** - PAL, guardrail, and objective configuration
- **tool-calling/** - Function calling with tools

Run examples with:

```bash
TAVUS_API_KEY=your-key go run examples/basic/main.go
```

## Development

### Regenerate API Client

The SDK is generated from the OpenAPI spec. To regenerate:

```bash
./generate.sh
```

This runs ogen and applies post-processing fixes from plexusone/ogen-tools.
