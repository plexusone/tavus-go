# Release v0.2.0

**Release Date:** 2025-07-06

This release makes the API types publicly accessible and adds custom LLM integration examples.

## Highlights

- **Public API Types** - API types moved from `internal/api` to `api` for external package integration
- **Multi-Provider LLM Support** - Custom LLM server example now supports OpenAI, Anthropic, X.AI, Ollama, and more via omnillm-core
- **Custom LLM Examples** - New examples demonstrating custom LLM backend integration with Tavus

## Breaking Changes

### Import Path Change

The ogen-generated API types have been moved from `internal/api` to `api`. Update your imports:

```go
// Before (v0.1.x)
import "github.com/plexusone/tavus-go/internal/api"  // No longer accessible

// After (v0.2.0+)
import "github.com/plexusone/tavus-go/api"
```

This change enables external packages (like omni-livekit) to use the SDK's type system directly.

## New Features

### Custom LLM Backend Examples

Two new examples demonstrate how to use Tavus with custom LLM backends:

**`examples/custom-llm/`** - Shows how to create a PAL with custom LLM configuration:

```go
pal, err := client.Pals().Create(ctx, &api.CreatePalRequest{
    PalName:       "Life Coach (Custom LLM)",
    DefaultFaceID: faceID,
    SystemPrompt:  api.NewOptString("Your system prompt here..."),
})
```

**`examples/custom-llm-server/`** - OpenAI-compatible chat completions server that can proxy to multiple LLM providers:

```bash
# OpenAI
LLM_PROVIDER=openai OPENAI_API_KEY=sk-... go run main.go

# Anthropic
LLM_PROVIDER=anthropic ANTHROPIC_API_KEY=sk-ant-... go run main.go

# Ollama (local)
LLM_PROVIDER=ollama LLM_MODEL=llama3:8b go run main.go
```

Supported providers via omnillm-core:

- OpenAI (gpt-4o-mini, gpt-4o, etc.)
- Anthropic (claude-3-5-haiku, claude-3-5-sonnet, etc.)
- X.AI (grok-3-mini, etc.)
- Ollama (llama3, mistral, etc.)
- Kimi, GLM, Qwen

## Upgrade Guide

### From v0.1.x

1. Update the module:

```bash
go get github.com/plexusone/tavus-go@v0.2.0
```

2. Update imports if you were using internal API types:

```go
// Change this:
import "github.com/plexusone/tavus-go/internal/api"

// To this:
import "github.com/plexusone/tavus-go/api"
```

3. If using the custom-llm-server example, install omnillm-core dependency:

```bash
go get github.com/plexusone/omnillm-core
```
