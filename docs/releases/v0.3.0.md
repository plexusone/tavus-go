# Release v0.3.0

**Release Date:** 2026-07-19

This release adds an OmniAvatar render adapter and publishes an interactive Scalar API reference.

## Highlights

- **OmniAvatar Render Adapter** - Use Tavus video generation behind the provider-agnostic [OmniAvatar](https://github.com/plexusone/omniavatar) interfaces
- **Interactive API Reference** - Scalar-based reference deployed to GitHub Pages at [plexusone.github.io/tavus-go](https://plexusone.github.io/tavus-go/)

## New Features

### OmniAvatar Adapter

The new `omniavatar` subpackage implements the `omniavatar-core` `render.Provider` interface on top of this SDK, so Tavus video generation can be used behind the OmniAvatar abstraction. It depends only on `omniavatar-core`, keeping Tavus-specific render knowledge in this repo.

```go
import tavusomni "github.com/plexusone/tavus-go/omniavatar"

p, err := tavusomni.NewRenderProvider(tavusomni.RenderConfig{
    APIKey: os.Getenv("TAVUS_API_KEY"),
})
```

Tavus has no audio upload API, so supply a publicly fetchable `render.GenerateRequest.AudioURL`. The real-time (live) CVI adapter, which requires LiveKit, lives in the batteries-included [`omniavatar`](https://github.com/plexusone/omniavatar) package.
